MIT Introduction to EECS I (6.01)教学智能车 

2018.5 - 2018.9
靳辉、巩力铜 

• 这是麻省理工学院 6.01 课程教学用车，由于该车已停产，学堂决定自行研发以对接教学内容。
• 使用 Python 编写控制智能车运动代码，根据前端 8 个声呐获取信息进行自动导航和避开障碍，并且根据贪心算法在自动探索之后绘制位置场地的地图。
• 项目历时3个多月，经过项目规划、代码编写、实验测试、Code Review，最后成功完成项目，能够完美地对接 MIT 课程，并且修复了原有的一些 Bug，改进了小车的结构。

AndroidWM: 轻量级安卓水印框架

2018.9.1 - 2018.9.17
黄以正 


• 一款轻量级 Android 开源图片水印框架，支持文字水印、图形水印和隐形数字水印。
• 隐形水印由空间域 LSB 算法实现，代码根据工厂者模式开放接口，利用 NDK 增加软件计算效率。
• 上榜 Github 全球热门 Java 项目周排行榜，现有 500+ star，超过全球8个国家的开发者使用。

智能电源

2018.8.19
崔洪伟 

• 该智能电源除去能双路输出可选的12V和5V两档直流电，还能对输出电流的参数进行实时测量，操作简捷并附有完整齐全的保护功能。
• 基于两块STC89C52单片机，单片机之间建立通信，相互协调，实现温度监控、负载调节显示、系统空闲保护、参数自动保存等功能。

智慧图书馆

2018
卢恒昌、张珈晖 


• 智慧图书馆是由崇新学堂学生开发的一款微信小程序客户端。该客户端主要为用户提供图书推荐，情感分析，猜你喜欢等功能，在读者选书的时候提供一定的参考。
• 另外，我们还针对找书难的问题，开发了图书定位功能，帮助读者能更快找到书籍。
